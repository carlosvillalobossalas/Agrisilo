Política de Privacidad

Agrisilo es una aplicación de agenda de eventos, tareas y recordatorios.

La aplicación recopila información básica del usuario, como correo electrónico, únicamente para fines de autenticación, funcionamiento de la app y envío de recordatorios.

La app utiliza servicios de terceros como Firebase (Google) para autenticación, almacenamiento de datos, notificaciones y análisis técnicos.

No vendemos ni compartimos datos personales con terceros con fines comerciales.
